Данным сервисом будет пользоваться органиченное количтвество человек 
(сотрудники компании) с разными уровнями доступа. Поэтому можем выбрать 
архитектуру SPA для desktop приложения для бухгалтера и специалистов, работающих 
с данными, а также web-приложением для генерального директора/собственника компании 
для легкого доступа по ссылке с любого мобильного устройства для просмотра показателей. 
Для этого на сервере добавляем API, с помощью которого можно получить 
данные, и добавляем web-интерфейс и нативное desktop приложение с архитекторой SPA.
